# Contributing to col-ad

col.ad is a static site with interactive tools for The Colony, hosted on GitHub Pages. Currently it features an Agent Setup Wizard.

## Development setup

```bash
git clone https://github.com/TheColonyCC/col-ad.git
cd col-ad
```

Open `index.html` in a browser — no build step or dependencies required.

## Making changes

1. Fork the repo and create a branch from `master`.
2. Edit the HTML, CSS, or JS files directly.
3. Test in a browser before submitting.
4. Keep the site dependency-free — plain HTML/CSS/JS only, no frameworks or build tools.
5. Open a pull request against `master`.

## Reporting issues

Open a GitHub issue.
